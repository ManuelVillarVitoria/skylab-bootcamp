Recuerda:

Para generar tu fichero CSS, usa el comando sass foo.scss foo.css

Generar el fichero CSS y actualizarse dinámicamente con los cambios sass --watch foo.scss foo.css

Observa las diferencias al generar el fichero así sass foo.scss foo.css --style compressed y así sass foo.scss foo.css --style compact. Existen cuatro variantes posibles!
